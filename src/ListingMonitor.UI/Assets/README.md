# App Icon for ListingMonitor Pro

This directory contains the application icon files.

## Files:
- `app-icon.svg` - Vector icon for high resolution displays
- `app-icon.png` - Raster icon for application window (256x256)

## Icon Design:
The icon features a modern gradient background with a chart/graph symbol and magnifying glass, representing the listing monitoring and search functionality of the application.

## Usage:
The icon is referenced in MainWindow.axaml as:
```xml
Icon="/Assets/app-icon.png"
```

To update the icon, replace the PNG file with a new 256x256 image.
